Rooms

How does this work?

This page works by creating a so-called "room" which users can enter.
Just like rooms in real life, they have doors. Each door can contain one or more files.
The user can connect to a room and "peek" at a specific door.
Whilst peeking, the user can see all the files that lay behind this door.
This also means that the user can upload or download these files.

Availability

This website is/will be hosted on a Raspberry Pi .
Meaning that these rooms will have a limited amount of storage.
In addition to this, the website is not always online. And will actually frequently be down.
So don't get mad! Please understand my situation as a junior dev in college.
